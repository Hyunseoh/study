h1~h6 제목 입력 시 사용(1--->6)(big->small)
p 하나의 문장을 입력할 때 사용

img 속성명은 src 속성값은 이미지 주소. 종료태그 필요x alt=이미지 없을 때 뜨게 할 값
a 하이퍼링크 속성명은 href 속성값은 주소
ul 기호로 된 리스트 ul>li
ol 순서 있는 리스트 ol>li
속성명은 보통 id나 class 이용
